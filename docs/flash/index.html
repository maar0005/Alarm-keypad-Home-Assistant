🔐 Alarm Keypad

Flash the factory firmware onto your device. Requires Chrome or Edge on desktop.

Connect the device to this computer via USB.

Click Install below. Select the correct serial port when asked. Needs Chrome or Edge. Safari and Firefox do not support Web Serial.

After flashing, the browser will ask you to configure WiFi. Enter your home WiFi credentials. Alternatively, power on the device and connect to the "Alarm Keypad Setup" hotspot from any phone.

Open Home Assistant. The device will appear automatically. Add it using the shared API key.

Send the 6-character pairing code shown on the device LCD to HA to activate the keypad. See the setup guide in the repository for the HA automation / Blueprint.
